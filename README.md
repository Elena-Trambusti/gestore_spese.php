# 💰 Gestore Spese Personalizzato - Elena

Questo progetto è una **Web Application Full-Stack** nata per semplificare il monitoraggio delle spese quotidiane. L'applicazione permette di gestire l'intero ciclo di vita del dato attraverso un'interfaccia intuitiva e una solida logica backend.

## 🚀 Caratteristiche principali
- **Operazioni CRUD:** Possibilità di inserire, visualizzare ed eliminare record in tempo reale.
- **Calcolo Dinamico:** Il sistema calcola automaticamente il totale delle spese presenti nel database.
- **Interfaccia Responsive:** Design pulito e professionale sviluppato con HTML5 e CSS3.
- **Persistenza dei Dati:** Gestione sicura tramite database relazionale MySQL.

## 🛠️ Tech Stack
- **Backend:** PHP 8.x
- **Frontend:** HTML5, CSS3 (Segoe UI Design)
- **Database:** MySQL / MariaDB
- **Ambiente di sviluppo:** XAMPP, Visual Studio Code

## 📦 Installazione e Utilizzo
1. Clonare il repository nella cartella `htdocs` di XAMPP.
2. Importare il file `database/gestione_spese.sql` tramite phpMyAdmin.
3. Avviare i moduli Apache e MySQL dal Pannello di Controllo XAMPP.
4. Accedere all'app tramite browser all'indirizzo `http://localhost/gestore/`.
